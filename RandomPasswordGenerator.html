Python Project: Random Password Generator

This Python script generates a strong password by combining letters, digits, and special characters.

Key Features:

  • User Input: Prompts the user to specify the password length (minimum 8 characters).
  • Character Composition:
    • 30%: Lowercase and uppercase letters.
    • 20%: Digits and punctuation marks.
  • Randomization: Uses random.shuffle() for unpredictable results.
  • Validation: Ensures input is numeric and at least 8 characters long.

How It Works:

  1. Prompts the user for the desired password length and validates the input.
  2. Generates character subsets (letters, digits, punctuation) based on the length.
  3. Shuffles and combines the characters into a randomized strong password.
  4. Outputs the password to the user.

Usage:

Run the script in Python, input the desired password length, and receive a strong password as output.
